Política de Privacidade

Informação base sobre recolha, uso e conservação de dados enviados através do site.

1. Responsável pelo tratamento

O responsável pelo tratamento dos dados enviados através deste site é o Barata Studio, através dos contactos disponibilizados no próprio site.

2. Dados recolhidos

Podem ser recolhidos nome, nome da empresa, email, telefone, tipo de projeto, faixa de investimento, prazo, conteúdo da mensagem e qualquer informação adicional enviada voluntariamente no formulário de contacto.

3. Finalidade

Os dados são usados apenas para estruturar o pedido inicial, preencher a mensagem aberta via WhatsApp e enquadrar comunicação relacionada com o projeto solicitado.

4. Conservação

Os dados são conservados apenas pelo período necessário para responder ao pedido e gerir contactos comerciais razoáveis, salvo obrigação legal em contrário.

5. Partilha de dados

Os dados não são vendidos nem cedidos a terceiros para fins de marketing. A comunicação iniciada através do formulário pode envolver plataformas técnicas necessárias ao envio da mensagem via WhatsApp.

6. Direitos do titular

Pode pedir acesso, retificação ou eliminação dos seus dados através dos contactos disponibilizados no site.
